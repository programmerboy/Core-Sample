About

This is a Sample Application created with .Net Core and Angular. Below is the list of technology, functionality, components etc. used.

  • .Net Core 3.x
  • Entity Framework (Core)
  • CORS (Cross Origin Resource Sharing)
  • Angular 8+
  • Saas (Syntactically Awesome Style Sheets)
  • Angular Material (Forms, Toolbar, Autocomplete etc.)
  • Angular Material Theming
