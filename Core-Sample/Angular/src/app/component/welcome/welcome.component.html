.Net Core and Angular Client/Server Sample.

Feel free to explore different sections of the site. To Download the code and run on your machine, please do the following.

  • Get the sample code from Git Hub
  • Create a new Database either in SQL Server Express or MSSQLLocalDB and name it "Samples".
  • Make sure your connection string in appsettings.json is correct one.
  • Download the missing packages (from Nuget) and build project.
  • On the Package Manager Console, run this command "Update-Database".
  • Open the Carriers.sql and Devices.sql in SSMS and run.
  • If all good, then press F5 in Visual Studio.
  • There are two ways to run Angular Client app. In Angular Folder, either run "ng-serve" , OR "npm run dev"
  • If ran using "ng-serve", then URL will be http://localhost:4200/.
  • If ran using "npm run dev", then URL will be https://localhost:44397/.
